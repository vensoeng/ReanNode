Introduction

Welcome to NGAlert

NGAlert enables developers to seamlessly display alerts and notifications in their web applications. Our mission is to provide a simple and customizable solution for handling alert messages in a modern web environment.

Reference

Global Config

Set defaults once at app startup:

ngAlert.init({
  duration: 4000,
  closeOnOverlay: false,
  labels: {
    success: 'បានជោគជ័យ',
    error: 'មានបញ្ហាបានកើតឡើង',
    warning: 'ការព្រមាន',
    info: 'ព័ត៌មាន'
  }
});
ngAlert.success('សូមស្វាគមន៍មកកាន់ NGAlert!');

NGAlert.success(msg, options)

Displays a success message with a green theme, typically used for completed actions. 

ngAlert.success('Success message!')

NGAlert.error(msg, options)

Displays an error message with a red theme to notify the user of a failure or issue. 

NGAlert.error('An error occurred!');

NGAlert.warning(msg, options)

Displays a warning message with an orange theme to alert the user of potential risks. 

NGAlert.warning('A warning occurred!');

NGAlert.info(message,title,type)

Displays an informational message with a blue theme for general updates or tips. 

NGAlert.info('Informational message!');

NGAlert.init(config)

Initializes the library with global configuration settings such as default duration or themes. 

NGAlert.init(config)

NGAlert.version

A read-only property that returns the current version string of the NGAlert library. 

NGAlert.version   // "0.1.0"

With custom title and options

Pass an options object to customize the alert title, visibility duration, and more. 

NGAlert.success('Invoice #1042 has been sent.', {
  title: 'Email Sent',
  duration: 4000
});

Generic show()

The base method for creating alerts, allowing you to specify the message, type, and options manually. 

NGAlert.show('Message text', 'success', { title: 'Done', duration: 3000 });

Dismiss programmatically

Manually close an alert instance or clear all active alerts from the screen using code. 

const alert = NGAlert.info('Loading…', { duration: 0 }); // duration:0 stays open
// later...
alert.dismiss();
// or dismiss whatever is currently shown:
NGAlert.dismiss();

Return Value

Every show() / shorthand call returns an instance object:

const instance = NGAlert.success('Hello world!');
instance.dismiss();    // close it early
instance.element;      // the root DOM element
