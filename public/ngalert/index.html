NG Alert

Lightweight Popup Notifications for Modern Web Apps

NG Alert is a library created to facilitate the creation of Popup Messages or Notifications for display to users (User Interaction) on websites.
Designs by vensoeng.

Try it live

Click the buttons below to see the notifications in action. The library handles queuing, stacking, and animations automatically.

example.js 
// Set defaults once at app startup
ngAlert.init({
  duration:4000,,
  closeOnOverlay: false,,
  labels: {
    success: 'Success',
    warning: 'Warning',
    error: 'Error',
    info: 'Info'
  },
});

ngAlert.success('You have successfully used NG Alert! (អ្នកបានប្រើប្រាស NG Alert ដោយជោគជ័យ)');
